It's All In The Style

After typing up some HTML, the next step would be to add some style to it. No end-user wants to view a page that doesn't have some character to it. Therefore, in the next portion of the tuturial, we shall add some CSS. It is recommended when typing up the CSS, to switch to the Splitscreen view, so that when modify the HTML elements, you can see the changes that you make. You can also switch to Design view to visually make your modifications.

Adding some CSS to our page!

Alright, so we decided that for the page we are creating we want something more than just black text with a white background. First off, select the CSS tab on Dreamweaver, and you will see a few elements from the HTML file already generated. Another option is to create a new CSS file, which you can do by going to File | New, however, this time select CSS under page type and create it. You will then a see a new CSS file waiting to be modified.

Picking up where we left off on the HTML elements, let's add some life to them. We'll begin with giving some properties to the body, such as background color, font type and color for the text. For those elements, give them the characteristics of a black background, with light green text color, and font style of Verdana, with additional font styles of Arial, Helvetica, sans-serif just in case the browser can't render the first option. It should resemble this: 


  #body {
  	background-color: #000;  		
  	font: Verdana, Arial, Helvetica, sans-serif;
    	color: #c1da9f; 
   }

The Content

Now to adjust how we view the content portion of the page, we'll add a dark green background, position the content within the page, and modify the header, which will coded as follows: 


  h1 {
   	font-size: 36px;
   	font-style: italic;
   	font-weight: bold;
   	margin-top: 50px;
   }
    
 .content {
  	margin-top: 50px;
  	margin-right: 20px;
  	margin-bottom: 50px;
  	margin-left:auto;
 	padding: 10px 0;
  	width: 80%;
  	background-color: #101908;
  }

The Footer

Lastly, we'll add some style to the footer, including repositioning it to the bottom of the page and adding a different color, style and size. 


    .footer {
    	font-size: 24px;
	font-style: italic;
	margin-top:50px;
	margin-left: 150px;
	padding: 10px 0;
	background-color: #0a100;
    }
