Scripting in an HTML doc

The last thing we'll cover is adding some scripts to the page, thus giving our web site some added features. First, I'll go over a bit of coding that was used in this tutorial, then we'll add it and another script to our Sample file. Before we can do that, however, go ahead and copy and paste the following code to the head of your page so that all of scripts we create can be rendered properly: 

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.0/jquery.min.js">
   </script>

Back to the Top

For the Jquery code used to bring the user back to the top of the page, we'll create a Javascript file named back2top, and call it from the HTML page.

To begin, go to File | New, then select Javascript file. Once created, add the following lines to it: 


   $(function() {
	$(window).scroll(function() {
		if($(this).scrollTop() != 0) {
			$('#toTop').fadeIn();	
		} else {
			$('#toTop').fadeOut();
		}
	});
 
	$('#toTop').click(function() {
		$('body,html').animate({scrollTop:0},800);
	});
    return this	
	})

After that, include the script source to the head and the div element to the bottom of the page and input the text below inside the div tags:

Script Source


	<script src="some.js" type="text/javascript"></script>

Near the bottom of the page


	<div id="toTop">^ Back to Top</div>

Once done, you should have some text floating when a user scrolls down, in which the function will scroll up to the top of the page. Remember to add some CSS to style it to fit the page.

Adding Content to the Page

Next, let's add some more text to the content using the onload function, planting it inside the body tag. 


    <body onLoad="appendText()">

Afterwords, open up the Some.js file and include this after the back2top function: 


	function appendText()
	{
	var txt1= $("<p></p>").text("This is some text that is 
        	generated through JQuery, using the onLoad object.");
            
	var txt2= $("<p></p>").text("Don't forget to scroll down 
   		so that you can see the Back2Top script in action.");
            
	$(".content").append(txt1,txt2);

	return this;
	}
