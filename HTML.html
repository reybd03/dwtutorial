  • Did You Know? → The DOCTYPE declaration defines the document type
    The text between <html> and </html> describe the web page.
    The text between <body> and </body> is the visible page content.
    The text between <h1> and </h1> is displayed as a heading.
    The text between <div> and </div> is a division within the page.
    The text between <p> and </p> is displayed as a paragraph.

The HTML

This page will cover things concerning the HTML portion of Dreamweaver, including how to properly set up a page from the start. The majority of this portion of the tutorial will be based on the Code and Splitscreen views, with a video covering the Design option.

What Happens Next?

To start off with, first select File | New, then proceed to create a new HTML file. Since this will be our first page, we will choose to create a new CSS as well. More on CSS creation in the next page.

When first opening a new HTML document there is only going to be a few codes on there: a skeleton to start your masterpiece. Automatically, Dreamweaver will include the document type, which in this instance is HTML 5; as well as the meta character-set, title, and body tags. 


    <!DOCTYPE HTML> 		
    <html>			
    <head>
    <meta charset="utf-8">	
    <title>Untitled Document</title>
    </head>
    
    
    <body>    
    </body>
    </html>

Setting Up The Head

Alright, so we have a fresh page waiting to be coded. Next, we are going to rename the title and include any CSS, Javascript, or any other files that we would like to be included as part of the page. Also, here is where we can include any style or scripts that we would to pretain to this page only. 


    <!DOCTYPE HTML> 		
    <html>			
    <head>
    <meta charset="utf-8">	
    <title>Sample</title>
    <link href="some.css">	
    <script type="text/javascript" src="some/scripts.js"></script>
    </head>
    <body>

Modifying the Body

Finally, we'll add some code to the body, so that when we load the page, it will show some text. To begin, set up div tags and nest some paragraphs within it. For later use with CSS and Javascript, add class and id attributes to the tags so that they can provide additional information to the content. Along side this example will be the Design portion. 


    
    <body>    
      <div class="content">
      <h1> The Header </h1>
        <p> Some Text </p>
         </div>
      <div id="footer"> 
      	<p>This is the footer<p>
       </div>
    </body>
    </html>
